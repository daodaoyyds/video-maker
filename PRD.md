# AI视频生成器 - 产品需求文档

## 项目概述
一个基于扣子工作流的AI视频生成工具，帮助用户快速生成符合行业要求的营销视频片段。

## 5步核心流程

### Step 1: 基础信息输入
| 字段 | 类型 | 必填 | 说明 | 示例 |
|------|------|------|------|------|
| 商品名称 | 文本 | ✅ | 视频中如何称呼产品 | "光感美白精华液" |
| 产品图片 | 文件 | ✅ | 本地上传产品图 | [上传图片] |
| 视频时长 | 单选 | 默认 8s | Cloudsway限制4/8/12秒 | 4s / 8s / 12s |
| 画面比例 | 单选 | 默认 9:16 | 竖屏/横屏/方形 | 9:16 / 16:9 / 1:1 |

### Step 2: 产品调研（对接扣子智能体）

#### 2.1 产品信息补充
- **输入**：用户提供的商品名称
- **调用**：扣子智能体1（productResearch）
- **输出**：
  - 基本信息：规格、参数、价格、适用人群
  - 核心技术
  - 核心功效
  - 解决痛点
  - 产品外观：形态、尺寸、颜色、材质、文字、场景

#### 2.2 TA画像调研
- **输入**：用户提供的商品名称
- **调用**：扣子智能体2（taResearch）
- **输出**：4个TA画像选项

#### 2.3 用户选择
- 选择TA画像
- 选择场景尺度（日常/特殊/极端）
- 选择情节尺度（轻度/中度/高度）

### Step 3: 脚本提案（对接扣子智能体）

#### 3.1 生成脚本提案
- **输入**：产品信息 + TA选择 + 场景/情节尺度
- **调用**：扣子智能体3（scriptProposal）
- **输出**：4个脚本提案
  - 标题、关系类型、人物设定
  - 大纲（钩子、痛点、解决、结尾）
  - 创意亮点

#### 3.2 用户选择
- 从4个脚本中选择1个

### Step 4: 提示词预览（对接扣子智能体）

#### 4.1 生成剧情脚本
- **输入**：选中的脚本 + 场景/情节尺度
- **调用**：扣子智能体4（scriptGeneration）
- **输出**：完整剧情脚本（Markdown格式）

#### 4.2 展示三部分内容
1. **剧情脚本** - 可编辑，可复制
2. **产品外观** - Step2数据转为Markdown
3. **固定参数** - 拍摄参数、技术要求

#### 4.3 用户操作
- 编辑任意部分
- 复制完整提示词
- 进入视频生成

### Step 5: 视频生成（对接Cloudsway）

#### 5.1 提交生成任务
- **输入**：完整提示词 + 视频参数（时长/比例）
- **调用**：Cloudsway API
- **输出**：视频任务ID

#### 5.2 轮询等待
- 每5秒查询状态
- 预计3-5分钟完成

#### 5.3 结果展示
- 视频预览
- 下载按钮
- 重新生成选项
