# AI视频生成器 - 产品需求文档

## 项目概述
一个基于扣子工作流的AI视频生成工具，帮助用户快速生成符合行业要求的营销视频片段。

## 核心功能流程

### 第一阶段：用户输入
| 字段 | 类型 | 必填 | 说明 | 示例 |
|------|------|------|------|------|
| 商品名称 | 文本 | ✅ | 视频中如何称呼产品 | "光感美白精华液" |
| 产品图片 | 文件 | ✅ | 本地上传产品图 | [上传图片] |
| 视频时长 | 数字 | 默认 15s | 视频总时长 | 15s / 30s / 60s |
| 画面比例 | 单选 | 默认 9:16 | 竖屏/横屏 | 9:16 / 16:9 / 1:1 |

### 第二阶段：产品调研（Prompt Agent 1）

#### 2.1 产品信息补充
- **输入**：用户提供的商品名称和图片
- **调用**：扣子智能体（搜索相关技能）
- **输出**：
  - 基本信息：规格、参数、价格、适用人群
  - 核心技术
  - 核心功效
  - 解决痛点

#### 2.2 TA画像调研和确认
- **输入**：用户提供的商品名称和图片
- **调用**：扣子智能体（小红书搜索技能）
- **调研流程**：
  1. 多关键词搜索
  2. 获取搜索结果
  3. 抓取正文内容
  4. 分析生成报告
- **输出**：TA画像建议（含多个选项）

#### 2.3 用户选择
- 选择TA画像
- 选择场景尺度 + 情节尺度

### 第三阶段：脚本提案（Prompt Agent 2）

#### 3.1 关系网络生成
基于TA画像和场景选择，生成关系网络矩阵：
- 关系类型 × 情节尺度 = 多个脚本方向
- 场景选择（日常/特殊/极端）

#### 3.2 生成脚本提案
- 每个关系至少1个脚本
- 4-6个脚本供用户选择
- 包含：大纲、人物设定、分镜设计

### 第四阶段：脚本增强（Prompt Agent 3）

#### 4.1 脚本丰富化
将简单脚本增强为标准视频生成提示词：
- 人物外貌描述
- 表情变化细节
- 动作细节
- 场景环境
- 台词优化

#### 4.2 风格描述
- 整体风格定位
- 拍摄质感
- 商业美感
- 情感氛围

#### 4.3 拍摄指导
- Camera设置
- Lens选择
- Lighting配置
- Mood设定

#### 4.4 分镜描述
- Shot 1、Shot 2、Shot 3...
- 每个镜头的时长、画面、动作

### 第五阶段：完整提示词生成（Prompt Agent 4）

#### 5.1 产品细节分析
调用多模态模型分析产品图片，输出：
- 产品类型
- 形态说明
- 尺寸比例
- 主色调
- 主要文字元素
- 材质质感
- 使用场景暗示

#### 5.2 提示词拼装
- 增强版脚本
- 产品细节
- 锁定要求

#### 5.3 提示词校验
- 长度校验（< 8000字）
- 违禁词校验

### 第六阶段：视频生成
- 调用视频生成模型
- 返回视频结果

## 扣子智能体调用节点设计

### 智能体1：产品调研Agent
```
输入：商品名称 + 产品图片
输出：产品信息JSON + TA画像选项
技能：搜索技能、小红书搜索技能
```

### 智能体2：脚本生成Agent
```
输入：产品信息 + TA选择 + 场景选择
输出：4-6个脚本提案
技能：创意写作技能、分镜设计技能
```

### 智能体3：脚本增强Agent
```
输入：选定脚本 + 视频参数
输出：增强版脚本（含风格、拍摄指导、分镜）
技能：视频提示词优化技能
```

### 智能体4：提示词组装Agent
```
输入：增强脚本 + 产品图片
输出：完整视频生成提示词
技能：多模态分析技能、提示词校验技能
```

### 智能体5：视频生成Agent
```
输入：完整提示词 + 视频参数
输出：视频文件URL
技能：视频生成API调用技能
```

## 网页功能模块

### 1. 首页/入口
- 项目介绍
- 开始创作按钮
- 历史项目列表

### 2. 创建流程（分步骤向导）

#### Step 1: 基础信息输入
- 商品名称输入框
- 产品图片上传（支持拖拽）
- 视频时长选择（15s/30s/60s）
- 画面比例选择（9:16/16:9/1:1）

#### Step 2: 产品调研结果展示
- 产品信息卡片展示
- TA画像选项卡片（可多选或单选）
- 场景尺度选择
- 情节尺度选择

#### Step 3: 脚本提案展示
- 脚本卡片列表（4-6个）
- 每个脚本包含：
  - 标题
  - 关系类型标签
  - 人物设定摘要
  - 情节亮点
  - 选择按钮

#### Step 4: 脚本详情展示
- 完整脚本内容
- 人物外貌描述
- 表情变化
- 动作细节
- 场景环境
- 台词

#### Step 5: 增强脚本展示
- 风格描述
- 拍摄指导
- 分镜描述（可视化时间轴）

#### Step 6: 完整提示词预览
- 可编辑的提示词文本
- 产品细节展示
- 生成视频按钮

#### Step 7: 视频生成与结果
- 生成进度条
- 视频预览
- 下载按钮
- 重新生成选项

### 3. 历史项目管理
- 项目列表
- 搜索和筛选
- 重新编辑
- 删除

### 4. 用户设置
- API配置（扣子API密钥）
- 默认参数设置
- 导出配置
